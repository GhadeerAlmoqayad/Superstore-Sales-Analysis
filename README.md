# 📊 Sales Analysis Project

## 📌 Project Overview

This project analyzes sales data to find useful insights about sales performance.

## 🎯 Objectives

* Analyze sales trends over time
* Identify top products
* Understand profit distribution

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Excel

## 📂 Dataset
From Kaggle.
The dataset includes:

* Order Date
* Product Name
* Sales
* Profit

## 📊 Key Insights

* Sales increase in certain months
* Some products sell a lot but have low profit
* A small number of products generate most revenue
